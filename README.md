## 📦 快速开始

### 环境准备

- Node.js 16+
- pnpm

### 安装步骤

```bash
# 克隆项目
git clone https://github.com:ZeroCatDev/Classworks.git
cd Classworks

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
